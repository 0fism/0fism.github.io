The website I have recently built, as of the week of January 13, 2024, serves as a starting point for showcasing my projects and experiences. While it may not fully demonstrate my abilities at this stage, I welcome you to reach out and learn more about me.

Through the website, I aim to provide companies with insights into my skills and expertise, with the hope of securing remote work job opportunities in the future. As time progresses, I will continue to expand the content, sharing more projects and accomplishments to further highlight my capabilities.

Streamlined Profile Showcase

This website is a simple and straightforward platform designed to showcase my previous project experiences, which have led me to my current field as a technical marketer. Utilizing Hexo, npm, and Hexo configuration files, I deployed the website to GitHub using Git commands. The main focus of this website is to provide a prompt and concise overview of my professional journey. It’s worth mentioning that the website doesn't involve frontend CSS or JS development, nor does it require any backend technical efforts. It is a static template using Markdown, with the Obsidian theme. Please note that the website does not prioritize SEO optimization. Thank you for visiting!”
